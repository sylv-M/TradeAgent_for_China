# China-Fund-Multi-Agent

为大学生散户投资者定制的专注于中国公募基金的交互式多 Agent 量化诊断与风险管控框架。

## 核心功能与基金透视

在执行净值量化诊断前，程序会基于公开披露资料进行全面的基金透视分析：

* **最新持仓与行业配置：** 详细解析最新披露期的前十大持仓及行业暴露。
* **持仓变动分析：** 对比上一报告期，追踪季度间的持仓比例变化。
* **基金经理画像：** 提供公开任职履历、在管总规模（AUM）及现任基金的最佳历史回报。
* **历史表现指标：** 包含基金成立以来收益、近一年/近三年收益、年化收益率及成立以来的最大回撤。

> **关于数据披露与局限性的说明：** 持仓与行业数据均来自定期监管报告，属于历史披露数据而非实时仓位。此外，基金的历史表现并不直接等同于基金经理个人的业绩归因。本功能仅用于风险暴露评估与信息管理——**绝不产生任何买卖指令或收益预测**。

## 针对中国大学生投资者的核心适配
原始的 *TradingAgents* 框架侧重于通过全球 API 对美股（如 AAPL、NVDA）进行基于 LLM 的规则交易，而本仓库针对中国高校学生的具体需求与财务边界进行了深度适配：

1. **锚定本土公募基金：** 将美股工具替换为本土化公募基金数据基础设施（`akshare`），支持对热门板块基金（如消费/白酒、半导体、医疗健康等）进行深入诊断。
2. **严谨的 MIT 评估指标：** 完全复刻原论文附录中指定的 portfolio 核心评估标准（公式 S1-S4），基于最近 90 个交易日计算实时量化指标：
   * **CR**（累计收益率 / Cumulative Return）
   * **ARR**（年化收益率 / Annualized Return）
   * **SR**（夏普比率 / Sharpe Ratio）
   * **MDD**（最大回撤 / Maximum Drawdown）
3. **结构化通信协议：** 在 Agent 之间强制执行论文中严格的 JSON-object 消息传递约束，有效消除 LLM 幻觉，并避免传统自然语言管道中常见的文本过载“传话筒效应”（Telephone Effect）。
4. **以学生为中心的安全风控护栏：** 配置多 Agent 辩论与风险管理节点，将抽象的量化波动率（特别是最大回撤 MDD）转化为亲切、易懂的“日常生活费”财务预警（例如结合典型的 2,000 元/月学生预算进行折算），防止冲动投资与本金受损。
